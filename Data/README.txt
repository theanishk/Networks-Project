Data used in this project is Citation-network V1 which has 629,814 papers and >632,752 citation relationships released in 2010.

Data is a text file with the following structure:
#* --- paperTitle
#@ --- Authors
#t ---- Year
#c  --- publication venue
#index 00---- index id of this paper
#% ---- the id of references of this paper (there are multiple lines, with each indicating a reference)
#! --- Abstract
